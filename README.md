## Day2(2020.9.21)
1.linux vim操作
>> esc 切换命令模式
>>> i  insert<br> 
>>> u  撤销一次操作<br> 
>>> Ctrl+r 恢复上一步被撤销的操作<br> 
>>>:w   保存文件但不退出vi<br> 
>>>:w!   强制保存，不推出vi<br> 
>>>:wq  保存文件并退出vi<br> 
>>>:wq! 强制保存文件，并退出vi<br> 
>>>:q  不保存文件，退出vi<br> 
>>>:q! 不保存文件，强制退出vi<br> 
>>>:e! 放弃所有修改，从上次保存文件开始再编辑<br>
